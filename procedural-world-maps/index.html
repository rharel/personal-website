@rharel Procedural World Maps

This is a collection of procedurally generated world bird's-eye-view maps. The generation process is a pseudo random physically based process following these steps:

  1. Generate terrain elevation using the diamond-square method
  2. Divide terrain into voronoi cells where each is associated with its own ground material (rock, sand, etc.)
  3. Compute temperature based on local altitude and latitude
  4. Place oceans where altitude is lower than zero
  5. Place ice where temperature is suitable for snowfall or where the oceans may freeze
  6. Compute precipitation based on local temperature, distance from the ocean, and climate zones
  7. Run rivers from local altitude maxima towards the ocean
