@rharel / Fluid Simulation
Either your browser does not support the canvas element, or you have JavaScript disabled. Both are required to view this project demo.

This is an implementation of two dimensional fluid simulation in WebGL based on the Real-Time Fluid Dynamics paper by Jos Stam. Move the cursor around to disturb the fluid or press to inject density at a given point. Source code here.
